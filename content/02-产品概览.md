## 产品概览
AI Portal 是企业级 LLM API 聚合与治理平台，提供 Free、Pro、Enterprise 三个版本，满足不同规模和需求的用户。

### 核心价值

- 零配置启动 - 注册即用，无需申请各厂商 API Key
- 统一计费 - 一个账户管理所有模型消费
- 灵活切换 - 随时切换不同厂商的模型
- 完整治理 - 从基础到企业级的治理能力

### 核心能力

1. LLM API 聚合
- 多协议支持 - OpenAI、Claude、Gemini 原生协议
- 主流模型 - GPT、Claude、Gemini、通义千问、豆包、Kimi 等
- 统一接口 - 一个 endpoint 访问所有模型

2. 治理能力

基础治理
- 用户组管理
- API Key 管理
- 基础权限控制
- 使用统计

高级治理

- 自定义业务空间
- 自主配置 Provider 和 Router
- 高级路由策略（负载均衡、故障转移）
- 完整审计追踪
- 成本分析和预算告警
- 安全治理（IP 白名单、配额管理、敏感信息过滤）
- 资源隔离（独立计算资源节点）
