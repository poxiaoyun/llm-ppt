## 产品版本
### 1. Free 版本 - 免费使用
适合：个人开发者、小团队、快速验证
核心功能：

- LLM API 聚合：通过消耗 Points 访问主流大模型，无需自己申请各厂商 API Key
- 基础治理能力：用户组管理、API Key 管理、基础权限控制
- 默认业务空间：使用统一的 endpoint（api.aportal.ai）
- 作用域支持：用户组级别、API Key 级别
- 付费方式：充值 Points，按实际消耗的 Token 数量扣费


### 2. Pro 版本 - 专业版
适合：中小企业、专业团队、生产环境

核心功能：
- 包含 Free 版本所有功能，额外提供：
- 自定义业务空间：创建独立的资源命名空间，拥有专属 endpoint
- 自主配置 Provider：使用自己的 API Key 对接各大厂商，成本透明
- 高级路由策略：负载均衡、故障转移、A/B 测试
- 完整权限管理：细粒度的访问控制和审计追踪
- 成本分析优化：多维度统计、预算告警
- 安全治理：IP 白名单、配额管理、敏感信息过滤
- 资源隔离：使用独立的计算资源节点
- 完整作用域支持：业务空间级别、路由级别、用户组级别、API Key 级别
- 使用方式：
- 可以继续使用默认业务空间（消耗 Points）
- 也可以创建自定义业务空间（使用自己的 API Key）
- 两种方式可以同时使用
- 付费方式：按月/年订阅 + Points 充值（可选）

### 3. Enterprise 版本 - 企业版
适合：大型企业、多业务线、严格合规要求

- 包含 Pro 版本所有功能，额外提供：
- 无限制自定义业务空间
- 企业级 SLA 保障
- 专属技术支持
- 定制化开发
- 私有化部署（可选）
